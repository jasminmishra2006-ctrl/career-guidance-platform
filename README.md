# One-Stop Personalized Career & Education Advisor

An AI-powered web platform designed to help students choose the right stream, course, and career path after Class 10/12.

## 🚀 Features
- Aptitude & Interest Assessment
- AI Chatbot & Personalized Q&A
- Course-to-Career Roadmap
- College Directory with Search & Filters
- Timeline Tracker for Admissions & Scholarships

## 🛠 Tech Stack
- Frontend: React.js, TypeScript
- Backend: Node.js
- Database: MongoDB, Supabase
- AI/ML: Python, TensorFlow, Scikit-learn

## 🎯 Goal
To provide personalized, data-driven career guidance and improve student decision-making.
